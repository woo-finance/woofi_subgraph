WOOFi Subgraph
======================

### Dependencies install
```shell
yarn install
```

### Deploy
```shell
graph auth --product hosted-service <Access Token>
graph codegen
graph deploy --product hosted-service <Github Username>/<Subgraph Name>
```

### Endpoints
  - bsc : https://api.thegraph.com/subgraphs/name/woonetwork/woofi
